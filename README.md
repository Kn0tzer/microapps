### [microapps.work.gd](https://microapps.work.gd)

microapps is a collection of small, handmade web apps built to be as simple as possible, even if it means compromising on some features. each app has a consistent, minimalistic user interface designed to be as easy to use as possible. everything is hosted on my vps, keeping it fast, secure, and reliable. sign in once and sync everything securely across your devices

/microtask is a simple to-do app for listing and checking off any tasks you need to remember. used as the foundation for all other microapps

/micronote is an unfinished, markdown based note-taking app

long term plans are to make a suite of single purpose utility apps. some microapps i plan to add, along with similar examples are: [local file converter](https://vert.sh), [file sharer](https://wormhole.app), [media downloader](https://cobalt.tools), [project organizer](https://excalidraw.com/), habit tracker, sleep tracker, [text saver](https://pastes.dev/), [bookmark tool](https://linkwarden.app/), and much more. something similar is [30tools](https://30tools.com/), except mine wont be super sketchy and vibecoded to death
