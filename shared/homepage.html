microapps

microapps is a collection of small, handmade web apps built to be as simple as possible, even if it means compromising on some features. each app has a consistent, minimalistic user interface designed to be as easy to use as possible. everything is hosted on my vps, keeping it fast, secure, and reliable. sign in once and sync everything securely across your devices
